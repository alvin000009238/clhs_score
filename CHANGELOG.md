# Changelog

## [1.1.2] - 2026-05-23

### Features
- 新增成績模擬器，可調整各科分數、採計科目與目標平均，快速試算調整後的平均成績。

### Bug Fixes
- 修正設定頁部分圖示底部被裁掉的問題。
- 修正科目頁展開多張成績卡片後，切到其他頁再回來時畫面跳動的問題。
- 提升成績模擬器入口卡片中「目前平均」與「科目」資訊的對比度，增加可讀性。
- 改善設定的操作穩定性。
- 改善返回、重新整理、設定、分頁與鎖定按鈕的辨識，提升輔助功能體驗。

### Performance Improvements
- 保留新版圓角圖示外觀，同時大幅縮小 App 體積。
- 優化成績模擬器滑桿操作，減少拖曳時不必要的震動回饋。

## [1.1.1] - 2026-05-22

### Features
- 新增「成績快取機制」：每次開啟 APP 時會優先瞬間載入先前查閱過的成績，無需每次重新等待網路請求，大幅提升瀏覽體驗，且支援離線查看。


## [1.1.0] - 2026-05-22

### Features
- 新增「設定」頁面：全新專屬設定選單，並將登出功能移至設定頁內。
- 支援外觀切換：可自由切換淺色模式、深色模式，或跟隨系統設定。
- 支援純黑背景：在深色模式下可啟用 AMOLED 純黑背景，提升夜間閱讀舒適度並節省電量。
- 支援動態色彩：Android 12 以上裝置支援 Material You 動態色彩，介面顏色會自動隨桌布變換。
- 內建更新檢查：可直接在設定頁面中連線 GitHub 檢查並取得最新版本。

### Bug Fixes
- 修復部分圖表與各科成績文字寫死顏色，使其在深色模式下能自動適應高對比度的正確顏色。

### Performance Improvements
- 優化圖表與介面元件的顏色渲染綁定機制，全面採用 Material Theme Token，提升動態主題切換時的繪製效能與穩定性。

## [1.0.1] - 2026-05-20

## 更新內容
- 全新登入體驗：內嵌學校網頁登入，並提供密碼自動填入（Autofill）支援
- 新增首頁介面，點擊後平滑切換至全螢幕 WebView 登入
- 徹底移除不穩定的傳統驗證碼登入機制
- 最佳化成績總覽畫面上方的操作圖示與樣式

## [1.0.0] - 2026-05-20

## 更新內容
- 優化卡片展開動畫
- 改善成績模擬器操作體驗
