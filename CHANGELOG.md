# Changelog

## [1.2.1] - 2026-06-20

### Bug Fixes
- 改善 WebView 登入流程的除錯資訊，當登入頁面資料解析失敗時更容易定位問題。
- 修正測試資料檢查，讓假資料驗證與最新資料格式一致。

### Performance Improvements
- 優化成績趨勢折線圖的點擊與繪製處理，減少互動時不必要的重複計算。
- 優化成績模擬器的歷史分數範圍計算，降低切換與重算時的額外負擔。
- 減少科目名稱與加權計算中的重複物件建立，提升大量成績資料處理時的效率。

## [1.2.0] - 2026-06-16

### Features
- 新增「生物識別鎖」功能：支援指紋解鎖保護 App，離開 App 後再回來時需重新驗證身分，防止他人偷看成績。
- 新增「段考資訊變更提醒」功能：開啟後 App 會在背景自動檢查成績是否有異動（如新公布成績、排名更新、五標變動等），有變化時推送通知提醒你查看。

### Bug Fixes
- 修正同時載入多筆成績時，偶爾出現 HTTP 401 的問題。
- 修正系統通知權限與 App 內通知開關狀態不同步的問題。

### Performance Improvements
- 大幅精簡安裝包體積，移除不必要的程式碼保留規則。
- 修正部分已知錯誤。

## [1.1.8] - 2026-06-11

### Features
- 優化首次開啟的介紹頁，改為多頁式導覽，讓主要功能更容易理解。
- 優化成績總覽頁，改用更精簡的底部導覽，並強化平均、總分、排名與重點分析的呈現。
- 優化成績模擬器，將自訂採計與目標反推拆成獨立區塊，並改善分數調整按鈕與滑桿操作。
- 優化成績折線圖與課表頁面，提升科目對比、課程詳情與不同螢幕尺寸下的閱讀體驗。

### Bug Fixes
- 修正成績匯出時排名欄位的格式一致性。

### Performance Improvements
- 整理多個成績頁面共用的排名、百分比與分布顏色邏輯，降低重複程式碼並提升維護性。
- 更新套件以提升穩定性。

## [1.1.7] - 2026-06-06

### Features
- 新增「匯出成績」功能，可將成績資料匯出為 CSV 檔案。
- 各科成績卡片新增顯示「校排」，並優化相關排名標籤的介面設計。
- 新增接收公告推播通知的功能，並提供開啟通知權限的提醒。
- 優化桌面課表小工具的使用體驗。

### Bug Fixes
- 修正部分已知錯誤。

## [1.1.6] - 2026-05-31

### Features
- 新增「成績折線圖」功能，可於進階頁面查看各科歷次考試成績的趨勢變化，並支援自由勾選、切換學期與多個科目對比，同時提供點擊成績點顯示詳細分數與自動置頂的功能。

## [1.1.5] - 2026-05-30

### Features
- 新增「我的課表」頁面，提供方便且流暢的查詢課表體驗，並支援下載課表圖片。
- 新增桌面課表小工具，可快速查看今日課程。

### Bug Fixes
- 修正快速切換考試時，較慢的舊成績請求可能覆蓋新選擇的問題。

### Performance Improvements
- 移除未使用的依賴，降低建置與封包負擔。
- 優化成績與課表快取解析流程。

## [1.1.4] - 2026-05-26

### Features
- 優化主頁排版：成績頁面向下滾動時，頂部標題與按鈕會自動滑出收起，最大化成績閱讀空間。
- 重新設計子頁面導航：設定、開發者選項與成績模擬器全面改用「浮動圓形返回按鈕」，並加入頂部漸層遮罩，讓穿越狀態列的內容能自然淡出，騰出更多畫面高度。

### Bug Fixes
- 修正成績模擬器中，卡片之間沒有保持標準間距的問題。

### Performance Improvements
- 加快各子頁面切換的動畫速度，大幅提升整體操作的俐落感與流暢度。
- 全面修復 Android 專案的 Lint 錯誤與警告，優化 Compose 狀態儲存機制，並升級 SDK 與核心相依套件至最新版本，進一步提升穩定性與效能。

## [1.1.3] - 2026-05-25

### Bug Fixes

- 修正科目頁展開多張成績卡片後，切到其他頁再回來時畫面跳動的問題。

## [1.1.1] - 2026-05-22

### Features
- 新增「成績快取機制」：每次開啟 APP 時會優先瞬間載入先前查閱過的成績，無需每次重新等待網路請求，大幅提升瀏覽體驗，且支援離線查看。


## [1.1.0] - 2026-05-22

### Features
- 新增「設定」頁面：全新專屬設定選單，並將登出功能移至設定頁內。
- 支援外觀切換：可自由切換淺色模式、深色模式，或跟隨系統設定。
- 支援純黑背景：在深色模式下可啟用 AMOLED 純黑背景，提升夜間閱讀舒適度並節省電量。
- 支援動態色彩：Android 12 以上裝置支援 Material You 動態色彩，介面顏色會自動隨桌布變換。
- 內建更新檢查：可直接在設定頁面中連線 GitHub 檢查並取得最新版本。

### Bug Fixes
- 修復部分圖表與各科成績文字寫死顏色，使其在深色模式下能自動適應高對比度的正確顏色。

### Performance Improvements
- 優化圖表與介面元件的顏色渲染綁定機制，全面採用 Material Theme Token，提升動態主題切換時的繪製效能與穩定性。
